How to Use the Tatkal Ticket Booking Extension

Fill all the form Login , Train and Payment Details

  1. Fill in your login details. Don't worry, your information is safe and stored securely in your Chrome local storage.
  2. Enter the details of the trains you want to book Tatkal tickets for. You can use the Go to IRCTC Website button to find the required information and fill it in the form.
  3. For now we have provided default payment options via Paytm UPI, which is faster compared to other payment methods.
  4. If you want to see the final QR code page for payment, check the box labeled Pay & Book (Show QR Code Page).
  5. Timer Details:
    1. Tatkal Start Timer:
      • If you select AC class, Tatkal start timer will be set for 09:59:53.
      • If you select Sleeper class, Tatkal start timer will be set for 10:59:53.
      • We recommend using the default timer settings.
    2. Refresh Time (ms): Default is set to 5000 milliseconds (5 seconds). This will refresh the train availability status every 5 seconds on the IRCTC page.
    3. Login Minutes Before: Default is set to 2 minutes. This will login at specified minutes before Tatkal start timer.
  6. The Tatkal Start Timer will be used to transition from search train page to train list page and there we are going to refresh the selected train using refresh time here 5 seconds
  7. Once you've filled all the details, click on the Save Settings button.

You can adjust both the start timer and refresh time according to your preferences and needs.

Auto Booking Switch

  1. If the switch is turned on, the extension will run automatically whenever you visit the IRCTC website and attempt to book tickets using the provided settings and passenger details.
  2. If the switch is turned off, the extension will not run on the IRCTC page.
  3. If the switch is on and Tatkal start timer is set, the extension will automatically log at specified minutes(2 minutes by default) before the start timer, and the Go to IRCTC Website button will disappear.
  4. If the switch is on and Tatkal start timer is set, and there is less than specified minutes available before the start timer, the Book Ticket on IRCTC button will appear, allowing you to book manually.

  • You can add multiple passenger by following above step
